DEPLOYMENT МЕТОДЫ:

1. Coolify (самый простой):
   - Используйте: docker-compose.coolify.yml
   - Документация: COOLIFY.md + COOLIFY_QUICKSTART.md
   - Время: 2 минуты

2. Docker Compose Local:
   - Используйте: docker-compose.production.yml + .env.example
   - Документация: QUICKSTART.md
   - Время: 1 минута

3. Bare Metal Linux:
   - Используйте: install.sh + provision.sh
   - Документация: DEPLOYMENT.md
   - Время: 5-10 минут

4. Kubernetes:
   - Используйте: Dockerfile.production + provision.sh
   - Документация: DEPLOYMENT.md (раздел Kubernetes)
   - Время: 10-15 минут

===========================================

QUICK START:

Coolify:
  1. Coolify → New Service → Docker Compose
  2. Repo → docker-compose.coolify.yml
  3. Set ENV in Coolify UI (FS_DOMAIN, EXTERNAL_SIP_IP, USERS, GATEWAYS)
  4. Deploy

Docker:
  1. cp .env.example .env
  2. nano .env (edit variables)
  3. docker-compose -f docker-compose.production.yml up -d
  4. docker-compose exec freeswitch fs_cli -x "sofia status"

Linux:
  1. sudo bash install.sh
  2. export FS_DOMAIN=... EXTERNAL_SIP_IP=... (etc)
  3. sudo bash provision.sh
  4. sudo systemctl start freeswitch
